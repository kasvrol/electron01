## Para estudar e rodar o código em seu computador 📚

* Clone este repositório
```bash
git clone https://github.com/kasvrol/electron01
```
* Acesse a pasta do projeto no terminal/cmd
```bash
cd ./electron01
```
* Instale as dependências
```bash
npm install
```
* Execute a aplicação 
```bash
npm start
```
